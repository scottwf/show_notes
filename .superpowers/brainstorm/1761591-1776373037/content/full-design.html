Design notes: • Week grid — Mon–Sun, today highlighted in blue, past days dimmed. ‹ / › navigate weeks, Today resets.
• Event chips — click navigates directly to episode detail page.
• Color coding — blue = episode, purple = premiere, red = finale.
• Lists below — existing TV Countdown, Season Finales, and Premieres sections unchanged.
• Data — events bucketed by date in Python, passed as JSON to Alpine.js for client-side week navigation (no extra requests).
• Filters — the existing filter tabs apply to both the grid and the lists.
